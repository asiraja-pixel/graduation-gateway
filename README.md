# Graduation Gateway (Monorepo)

This repo contains:

- `frontend/`: Vite + React + TypeScript UI
- `backend/`: Node.js API (TypeScript)

## Quick start

Install dependencies for both workspaces:

```sh
npm install
```

Run the frontend:

```sh
npm run dev:frontend
```

Run the backend:

```sh
npm run dev:backend
```

## Folder structure

```
frontend/   # existing UI app
backend/    # API service
```
